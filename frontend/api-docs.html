API Documentation

POST

/api/v1/scan

Uploads an image for disease detection and classification. Returns detailed probabilities and processing latency.

Request (cURL)

curl -X POST http://localhost:8000/api/v1/scan \ -H "accept: application/json" \ -H "Content-Type: multipart/form-data" \ -F "file=@leaf.jpg"

Response Example

{ "scan_id": "9b1deb4d-3b7d-4bad-9bdd-2b0d7b3dcb6d", "defect_class": "Tomato_Early_blight", "confidence": 0.9472, "probabilities": { "Apple_scab": 0.002, "Tomato_Early_blight": 0.9472, "Healthy": 0.012, "...": "..." }, "latency": { "preprocessing_ms": 3.2, "inference_ms": 28.5, "postprocessing_ms": 1.1, "total_ms": 32.8 }, "timestamp": "2026-06-23T08:30:00Z" }
